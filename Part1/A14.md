# AI-powered video surveillance 

Detects unusual behavior in real time alerting an admin to it while it is
occuring, making the surveillance system more of a preventative measure 
compared to a reactive measure.

# Behavioral biometrics systems

analyze typing/mouse patterns for identity allowing for real time verification
without the need to request some form of input to verify.

# AI threat detection platforms

Identify cyber threats using pattern analysis such as antivirus software or IDS
systems preventing malware which may differ from known malware or detecting 
suspicious user activity.

# Fraud detection systems

Flag suspicious transactions through AI to detect fraud without relying on 
detection of common fraud practices.

# Autonomous security robots/drones

Patrols areas and detect anomalies for a quick response to possible crimes or
issues.
