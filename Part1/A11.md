# Fingerprint scanner

Uses the unique fingerprint of an individual to 
control access based on whether that person is permitted
into an area or system

# Facial recognition 

Similar to fingerprint scanners but instead uses a persons
face to control access to the area or system

# RFID card reader 

RFID uses an antenna in a card or fob to recieve signals from
a RFID reader, it will then respond using through the antenna
with some form of code or identification. Which can be used to
control access.

# PIN entry

Uses a number entry to recieve a code which can then be used
to control access.

# Iris scanning

Uses the extremely unique pattern of the iris to determine
someones identity in a similar fashion to facial or fingerprint
recognition.
