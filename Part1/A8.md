# AES

AES is a symmetric key cipher, which means that encryption and decryption of messages
is performed with the same key unlike asymmetric key ciphers which have differing keys.
AES uses a 4x4 array to perform its encryption and decryption, where the text is
transformed by the array several times based on the size of the key, such as 10 for
128-bit, or 12 for 192-bit.
