# Firewall software 

Monitors and controls network traffic either through the router
or the computer itself.

# Antivirus software

Detects and removes malicious software through various methods
like consulting security vendors or analysing the machine code for
common patterns that match known malware.

# Password manager.

Securely stores and generates passwords to ensure a variation of 
passwords between sites to prevent credential stuffing.

# VPN (Virtual Private Network)

Encrypts internet traffic preserving the security of potentially 
viewable data from an ISP or a man in the middle.

# Intrusion Detection System (IDS)

Identifies suspicious network activity in order to avoid potential
attacks before or while they are occuring.
